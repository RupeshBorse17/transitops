# TransitOps 🚚

TransitOps is a Smart Transport Operations Platform developed for the Odoo Hackathon 2026.

## Features
- Vehicle Management
- Driver Management
- Trip Management
- Maintenance Tracking
- Fuel & Expense Tracking
- Dashboard & Reports
- Role-Based Access Control (RBAC)

## Tech Stack
- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- React.js
- Git & GitHub
